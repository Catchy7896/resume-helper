# 简历填写助手 Chrome 扩展

一个简洁易用的 Chrome 扩展，用于快速读取 Markdown 简历内容并管理投递记录。

## 功能特点

- 📄 上传 Markdown 格式的简历文件
- 🏷️ 自动解析标签列表，点击即复制到剪贴板
- 💾 自动保存上传结果，下次打开无需重复上传
- 🔄 支持重新上传，快速更新内容
- 📏 支持拖拽调整弹窗大小，记住上一次尺寸
- 📝 提供待投递/已投递列表，支持添加、编辑、切换状态、删除
- 🔗 投递记录支持录入日期、链接和备注

## 安装方法

1. 打开 Chrome，访问 `chrome://extensions/`
2. 打开右上角的“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目目录
5. 安装完成后，点击工具栏图标即可使用

## Markdown 文件格式

使用以下格式编写简历内容（标签名请包含在中括号内）：

```markdown
[姓名]
张三

[电话]
13800138000

[工作经历]
2020.01 - 至今 | XX公司 | 软件工程师
负责核心业务开发...
```

标签名与内容之间可以有空行，内容支持多行文本。

## 使用指南

### 简历内容
1. 首次使用点击“上传 Markdown 文件”选择简历文件
2. 上传成功后会显示所有标签，点击即可复制对应内容
3. 如需更新，点击“重新上传文件”重新选择 Markdown

### 投递记录
1. 切换到“投递记录”标签页
2. 点击“+ 添加记录”填写公司/职位、日期、链接（可选）、备注（可选）
3. 默认添加到当前页签（待投递/已投递），可在记录上点击 ✓/↩ 切换状态
4. 点击记录可进入编辑，支持修改状态、更新信息或删除

### 窗口调整
- 拖拽弹窗右下角斜纹区域可调整大小
- 关闭后再次打开会记住上一次的宽高

## 文件说明

- `manifest.json`：扩展配置
- `popup.html`：弹窗结构
- `popup.css`：界面样式
- `popup.js`：业务逻辑
- `example.md`：示例 Markdown 文件
