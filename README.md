# 简历填写助手 Chrome 扩展

一个简洁易用的 Chrome 扩展，用于快速读取 Markdown 简历内容并管理投递记录。可在填写过程中实时更新填写内容，支持投递记录，助力网申填写！

## 功能特点

### 核心功能
- 📄 上传 Markdown 格式的简历文件
- 🧩 自动识别模块与子项，分块展示所有内容
- 🖱️ 点击任意字段或条目即可复制到剪贴板
- ✏️ **支持在线编辑简历内容**：添加、修改、删除模块和条目
- 📥 **导出 Markdown 文件**：将编辑后的内容导出为 .md 文件
- 💾 自动保存上传结果，下次打开无需重复上传
- 🔄 支持重新上传，快速更新内容

### 窗口模式
- 📌 **固定窗口**：在新标签页中打开，可保持打开状态，不会因点击其他页面而关闭
- 📏 支持拖拽调整弹窗大小，记住上一次尺寸

### 表单填写
- 🎯 **智能填写表单**：点击"填写"按钮，自动填写到当前页面的表单字段
- 🎯 点击内容项自动复制到剪切板
- 🎯 优先填写当前聚焦的输入框


### 投递记录
- 📝 提供待投递/已投递列表，支持添加、编辑、切换状态、删除
- 🔗 投递记录支持录入日期、链接和备注

## Markdown 文件格式

推荐使用“模块 + 内容”或“模块 + 子项”的结构，示例如下：

```markdown
[基本信息]
姓名：张三
电话：13800138000
邮箱：zhangsan@example.com
个人简介：
- 5 年开发经验，专注前端与服务端协作
- 主导过千万级用户产品的性能优化

[工作经历-XX 科技有限公司]
公司名称：XX 科技有限公司
职位名称：高级工程师
在职时间：2020.01 - 至今
职责描述：
- 负责核心业务系统的架构设计与演进
- 搭建组件库，提升团队交付效率
```

规则说明：
- `[模块名]` 表示一个主模块（例如“基本信息”、“教育经历”），内部支持键值对、列表等内容
- `[模块名-子项名]` 可将同一模块下的不同经历拆成独立卡片（例如多个“工作经历”）
- 行内使用 `键：值` 或 `- 列表项`、`1. 列表项`，都会被识别为可单独复制的条目

## 使用指南

### 简历内容

#### 上传和查看
1. 首次使用点击"上传 Markdown 文件"选择简历文件
2. 上传成功后会按模块展示所有字段，点击任意标签或段落即可复制内容
3. 如需更新，点击"重新上传文件"重新选择 Markdown

#### 编辑简历内容
❗️更新：支持实时更新简历，方便在填写过程中需要更新一些新的填写内容/修改原内容。
- **添加模块**：点击"📥 导出"按钮旁的"+ 模块"按钮，输入模块名称和子项名称（可选）
- **添加条目**：点击模块或子项旁的"+"按钮，输入标签和内容
- **编辑条目**：点击条目旁的"✏️"按钮，修改标签和内容后保存
- **删除条目**：点击条目旁的"❌"按钮，确认后删除
- **删除模块**：点击模块旁的"❌"按钮，确认后删除整个模块

#### 导出 Markdown
- 点击"📥 导出"按钮，自动下载当前简历内容的 Markdown 文件
- 文件命名格式：`resume-YYYY-MM-DD.md`
- 导出的文件包含所有编辑后的内容，可直接使用

### 窗口模式

#### 固定窗口（标签页模式）
1. 点击右上角的"📌"按钮
2. 会在新标签页中打开固定窗口
3. 该窗口会一直保持打开，即使点击其他标签页也不会关闭
4. 适合需要频繁使用简历内容的场景（可使用**win+方向键**分屏填写更便捷）

### 表单填写

#### 方式一：复制粘贴
1. 点击字段名，内容会自动复制到剪贴板
2. 在目标表单中粘贴（Ctrl+V / Cmd+V）

#### 方式二：直接填写（推荐）
1. 在要填写的输入框或可编辑区域点击，使其获得焦点
2. 点击简历字段旁的"填写"按钮
3. 内容会自动填写到当前聚焦的字段中
4. 如果没有聚焦字段，会自动找到第一个空的输入框并填写

**提示**：
- 支持普通表单字段（input、textarea）
- 支持富文本编辑器（contenteditable 的 div、span 等）
- 如果填写失败，会自动回退到复制到剪贴板

### 投递记录
1. 切换到"投递记录"标签页
2. 点击"+ 添加记录"填写公司/职位、日期、链接（可选）、备注（可选）
3. 默认添加到当前页签（待投递/已投递），可在记录上点击 ✓/↩ 切换状态
4. 点击记录可进入编辑，支持修改状态、更新信息或删除

### 窗口调整
- 拖拽弹窗右下角斜纹区域可调整大小
- 关闭后再次打开会记住上一次的宽高

## 文件说明

### 核心文件
- `manifest.json`：扩展配置文件，包含权限和配置信息
- `popup.html`：弹窗页面结构
- `popup.css`：界面样式文件
- `popup.js`：主要业务逻辑，包括简历管理、窗口管理、表单填写等
- `background.js`：后台服务，管理固定窗口和悬浮窗的生命周期
- `content.js`：内容脚本，注入到网页中实现表单自动填写功能

### 示例文件
- `example.md`：示例 Markdown 文件，展示简历格式

## 技术特性

- **Manifest V3**：使用最新的 Chrome 扩展 API
- **本地存储**：使用 Chrome Storage API 保存简历数据和窗口设置
- **智能表单识别**：自动识别各种类型的表单字段和可编辑元素
- **事件触发**：正确触发 input/change 事件，兼容 React、Vue 等框架
- **异步消息传递**：使用 Chrome Runtime API 实现 popup 和 background 之间的通信
