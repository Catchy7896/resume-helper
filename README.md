# 简历填写助手 Chrome 扩展

一个简洁易用的Chrome扩展，用于快速复制简历内容到剪切板。

## 功能特点

- 📄 上传Markdown格式的简历文件
- 🏷️ 自动解析并显示标签列表
- 📋 点击标签一键复制内容到剪切板
- 💾 自动保存，无需重复上传
- 🔄 支持重新上传更新内容

## 安装方法

1. 打开Chrome浏览器，进入扩展管理页面：`chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本文件夹
5. 扩展安装完成！

## Markdown文件格式

使用以下格式编写你的简历Markdown文件：

```markdown
[标签名1]
内容1

[标签名2]
内容2

[标签名3]
内容3
```

### 示例

```markdown
[姓名]
张三

[电话]
13800138000

[邮箱]
zhangsan@example.com

[工作经历]
2020.01 - 至今 | XX公司 | 软件工程师
负责核心业务开发...
```

## 使用方法

1. 点击浏览器工具栏中的扩展图标
2. 首次使用：点击"上传Markdown文件"按钮，选择你的简历文件
3. 上传后，会显示所有标签按钮
4. 点击任意标签，内容会自动复制到剪切板
5. 如需更新内容，点击"重新上传文件"按钮

## 注意事项

- 扩展使用 emoji 图标，无需额外图标文件
- Markdown文件必须使用 `[标签名]` 格式定义标签
- 标签名和内容之间可以有空行
- 支持多行内容

## 文件说明

- `manifest.json` - 扩展配置文件
- `popup.html` - 弹出窗口界面
- `popup.js` - 主要功能逻辑
- `popup.css` - 样式文件
- `example.md` - 示例Markdown文件
